# Bridge Dental AI — Website

Static marketing site (plain HTML + assets). No build step.

## Pages
- `index.html` — home
- `about.html`, `features.html`, `pricing.html`, `pricing-simple.html`, `contact.html`, `earlyaccess.html`

## Deploy (Vercel)
Framework preset: **Other**. Build command: none. Output directory: `./` (root).
